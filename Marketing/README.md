# Marketing Campaigns Copilot

Answers campaign performance/ROI questions and captures structured campaign brief requests as Planner tasks.

**Guide:** `Marketing_Campaigns_Copilot_Full_Build_Guide.pdf`
**Sample data:** `Marketing_Campaigns_Data.xlsx`

## Workloads used

- Microsoft Copilot Studio
- Microsoft Planner
- SharePoint Online (optional record)
- Microsoft Teams

## Licenses required

- Copilot Studio
- Microsoft Planner (Microsoft 365)
- SharePoint Online
- Microsoft Teams

## Permissions required

- Environment Maker on the target environment
- Owner or member on the Marketing Campaign Requests Planner plan
- Permission to post to the marketing team's Teams channel

## Connectors used in this build

- Planner
- SharePoint
- Microsoft Teams

> **Note:** Create the Planner plan (with a bucket per region) before Step 6 — the flow needs somewhere to file the task.
