# Patient Intake Summariser Copilot

Turns free-text intake notes into structured pre-consult summaries and files clinician follow-up tasks.

**Guide:** `Patient_Intake_Summariser_Copilot_Full_Build_Guide.pdf`
**Sample data:** `Healthcare_Intake_Data.xlsx`

## Workloads used

- Microsoft Copilot Studio
- Microsoft Planner
- Microsoft Teams

## Licenses required

- Copilot Studio
- Microsoft Planner
- Microsoft Teams

## Permissions required

- Environment Maker on the target environment
- Owner or member on the Clinical Review Tasks Planner plan
- Permission to post to the clinician's Teams chat

## Connectors used in this build

- Planner
- Microsoft Teams

> **Note:** Data in the sample workbook is synthetic and pseudonymised (e.g. PT-014) — no real PII. Still confirm this fits your organisation's health-data handling policy before a live session; do not enable public web channels for this agent.
